# Portal GAS (clasp-managed)

このリポジトリは Google Apps Script (GAS) で作られた社内ポータルアプリです。

## 概要
- clasp で管理される GAS プロジェクト。
- WebApp はドメイン公開（`appsscript.json` の `webapp.access: DOMAIN`）。

## 目的
- 案件管理（List）/ タスク / 予定（Plan） / ルール / リンク集 / 稼働情報 / マニュアル を統合するポータル。

## ファイル命名規則（現行ルール案）
- 4 桁の数値プレフィックスで機能別に整理します。
  - `0XXX` - サーバー側（GAS）スクリプト（例: `0001_gs_initial.js`）
  - `10XX` - CSS 断片（HTML ファイルに埋め込むスタイル、例: `1000_css_main.html`）
  - `20XX` - HTML テンプレート（例: `2000_html_main.html`）
  - `30XX` - クライアント JS（HTML 断片、例: `3000_js_main.html`）
  - `99XX` - ユーティリティ / 生成スクリプト（例: `9999_index生成.js`）

- 命名は「プレフィックス_機能名.拡張子」を基本とする。
- 一時的な作業ファイルはプレフィックス無しで `無題.js` や `編集用.js` のように保存し、整理時に `archive/` へ移動または削除する。

## 開発手順（簡易）
- 変更をローカルで編集 → `clasp push` → スクリプトエディタでテスト → デプロイ。

## 既知の改善点 / 次の作業
- `README.md` を充実化（デプロイ手順、権限、主要シート名、設定の説明）
- コード整形ルール（ESLint/Prettier）導入検討
- 一時ファイルを `archive/` に移動するスクリプト作成

## 一時ファイル（扱い）
- `無題.js`, `編集用.js` は一時ファイルのため整理対象外（必要なら `archive/` に移動）。
